Learning Piano 7

Mar 22, 2017 | Hits

今天去六弦琴行练琴。感觉有点沮丧。

只练习了一小段，但是指法比较复杂，都是我不太熟练的。所以只练了半段，主旋律在右手上的那段。

练习曲是3/4拍，有一个动作我很不熟练：

对我来说，难的动作有：

  • 第一节，始终不适应
  • 第三节，还好，很快就记住了，配合第四节
  • 第五节，动作能记住，但是有时候会按错手指，不熟
  • 第六节，注意和第一节很像，但是左手的动作不一样，是分开弹两次的
  • 第八、九节，如上所述，对比更明显。

注意这里至少有4类我不熟悉的动作。今晚弹得很不愉快。

不过看看前面弹奏的曲子Learning Piano 4，就知道自己有所进步，毕竟没有挑战，怎么可能成长？

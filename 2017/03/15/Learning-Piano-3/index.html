Learning Piano 3

Mar 15, 2017 | Hits

今天继续去“六弦琴行”练琴，把昨天那一段左右手没有协调好的部分终于弹好了，有一点收获。由于右手的部分比较复杂，所以我先练好了右手，左手再配合右手同时演奏，花了很长时间。

对于肌肉记忆还是比较敏感，我通过手的动作识别出来和之前重复的部分，对于熟悉的动作，可以比较连贯的弹奏出来。我想，这样的积累，便是钢琴演奏经验积累的过程吧。

今天的进度慢了一些。主要是左右手协调的部分比较困难。

关门的小姐姐来催我走。问我练了多久钢琴，我让她猜，她猜30节课，然后又猜10节课。我觉得她实在是高看我了，其实我才上了一节课，外加自己练习了3节课呀。

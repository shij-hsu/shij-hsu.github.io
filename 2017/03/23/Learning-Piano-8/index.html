Learning Piano 8

Mar 23, 2017 | Hits

今天去了六弦琴行练琴。和昨天一样只练了半段，两次才把一首完整的曲子弹完。

找到了进度缓慢的原因——指法。

我的无名指比小拇指长不少，所以弹琴的时候不能很好的切换。我的领悟是，要想迅速地掌握动作，必须有一个良好的指法。即手指能够在无意识状态下放置到正确地位置上，确保各个手指彼此之间的独立性，不会带键，并且保证力度，声音能够控制。

这并不是一个很好达到的目标，今天练习的主旋律在左手上，右手只有食指和小拇指需要动，我注意到了无名指、中指放置的位置都不对，原因是我不需要用到，所以便忽视了。还有左手的拇指，刚开始的时候老是在某一小节卡住，后来我总结原因，发现是大拇指在不用的时候并没有也放在正确的位置上，导致要使用的时候来不及反应。

还有节奏也有一些小问题，左右手有时候没有配合好，没有同时奏出一个和弦。另外，折腾了一个多小时，手指很酸痛。

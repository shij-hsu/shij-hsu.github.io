Learning Piano

Mar 14, 2017 | Hits

上个星期六正式学琴，因为有一定理论基础，不是特别吃力，主要问题集中在对手指的控制方面。我的手比较大，所以拇指的工作常常不自觉让食指代劳了:)。另外对于“五线谱-（音高，节拍）-手指动作”的自然反射还未建立，常常出错。

今天周一，去琴行练了一下琴，对钢琴的感觉好多了。有几个感受：

  • 节拍器很重要，但是前提是正确地使用，让自己的节奏和节拍器合拍。
  • 音感很重要
  • 弹错了音不需要重头来，接着弹就是了

今天弹了两句4/4拍和3/4拍，感觉还可以:)，明天还要去琴行弹琴。

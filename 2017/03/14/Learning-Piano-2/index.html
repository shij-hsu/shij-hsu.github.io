Learning Piano 2

Mar 14, 2017 | Hits

今天去华韵琴行练了一会儿琴，从晚上7点一刻到8点20左右。去的时候小姐姐已经认出我来了（我这个脸盲真的很佩服她记人的本领）。有一个老师在上课，还有一个人也在练琴（或者吉他？），练的曲子是《天空之城》。

我先把昨天练的重复弹了一遍，今天弹了一段双手异步的练习。刚开始没有思路，后来是自己领悟，找到了一种感觉。弹琴不是靠逻辑思维，人的脑子不是计算机，不可能反应那么快；而是要靠直觉。比如，和节拍器的合拍，我相信这是高等动物独有的神奇技能，可意会不可言传。还有左手什么时候停止，右手什么时候松开，什么时候一起弹奏，总之，这些都是一种感觉。

弹琴是一个沉浸式的体验，身处其中的人基本上感受不到时间的流逝，也不会厌倦，我感觉到时间过得飞快，有点意犹未尽的感觉。也可能还没有进入瓶颈期吧，目前学得还是比较容易的。
